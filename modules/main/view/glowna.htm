Show your code

Create and embed working code samples. The main script will be automatically executed so everyone can see the results directly on your website.

Collaborate

Give others access to your sandbox and create the code together.

You can leave your comments and get comments from the other users.

Clone

Want to quickly try new approach without loosing existing work? Simply press Clone button and copy your sandbox with all the files and settings.

Isolate code

Every time code is executed it is sandboxed so it won't be affected by other's code.

You can lock your sandbox and give an access to selected users only.

Use any device

Desktop, laptop, tablet or mobile. We know that coding on mobile is not what the most people do, but still, mobile is all you need to get the sandbox and test new idea on the go.

PHPfy is designed to make editing and executing code on your smartphone as easy as on your home pc with 26" screen.
